# SizeRandomBot - Telegram бот для проведения розыгрышей

Профессиональный Telegram-бот для создания и проведения честных розыгрышей с проверкой подписок на каналы.

## 🎯 Основные возможности

- **Создание розыгрышей** с пошаговым конструктором
- **Проверка подписок** на обязательные каналы
- **Автоматическое подведение итогов** по расписанию
- **Честный выбор победителей** с помощью random.sample
- **Поддержка медиа** (фото и видео) в постах розыгрышей
- **Админ-панель** со статистикой и рассылкой
- **Управление розыгрышами** через удобный интерфейс

## 📋 Требования

- Python 3.9+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## 🚀 Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите:

```env
# Токен бота от @BotFather
BOT_TOKEN=your_bot_token_here

# ID администратора бота (ваш Telegram ID)
ADMIN_ID=123456789

# Использовать SQLite (True) или PostgreSQL (False)
USE_SQLITE=True
```

**Как узнать свой Telegram ID:**
- Напишите боту [@userinfobot](https://t.me/userinfobot)
- Или используйте [@getmyid_bot](https://t.me/getmyid_bot)

### 3. Запуск бота

```bash
python bot.py
```

## 📖 Использование

### Для пользователей

1. **Создание розыгрыша:**
   - Нажмите кнопку "🎉 Создать розыгрыш"
   - Следуйте инструкциям бота:
     - Отправьте описание конкурса (можно с фото/видео)
     - Укажите текст кнопки участия
     - Укажите количество победителей (1-100)
     - Добавьте каналы для обязательной подписки (до 5)
     - Выберите время завершения

2. **Публикация розыгрыша:**
   - Бот сгенерирует пост с кнопкой участия
   - Перешлите этот пост в свой канал
   - Участники смогут нажимать кнопку для участия

3. **Управление розыгрышами:**
   - Используйте кнопку "📊 Мои конкурсы"
   - Просматривайте статистику участников
   - Завершайте розыгрыши вручную при необходимости

### Для администратора

Администратору доступна кнопка "👑 Админ-панель" с функциями:

- **📊 Статистика** - общая статистика бота
- **📢 Рассылка** - отправка сообщений всем пользователям

## 🔧 Технологии

- **aiogram 3.x** - асинхронный фреймворк для Telegram Bot API
- **aiosqlite** - асинхронная работа с SQLite
- **APScheduler** - планировщик задач для автоматического завершения розыгрышей
- **python-dotenv** - управление переменными окружения

## ⚙️ Механика работы

### Проверка подписок

При нажатии на кнопку участия бот:
1. Проверяет подписку пользователя на все указанные каналы через `bot.get_chat_member`
2. Если пользователь не подписан - показывает список каналов с кнопками подписки
3. Если подписан - добавляет в список участников и обновляет счетчик

### Выбор победителей

При завершении розыгрыша:
1. Бот получает список всех участников из БД
2. С помощью `random.sample` выбирает N случайных победителей
3. Повторно проверяет подписки победителей на каналы
4. Исключает отписавшихся и выбирает новых
5. Обновляет пост в канале со списком победителей
6. Уведомляет создателя розыгрыша

### Автоматическое завершение

APScheduler каждую минуту проверяет активные розыгрыши:
- Если время завершения наступило - автоматически подводит итоги
- Обновляет пост в канале
- Отправляет уведомление создателю

## 📝 Важные замечания

1. **Бот должен быть администратором** в каналах-спонсорах для проверки подписок
2. **Права администратора** должны включать возможность просмотра списка участников
3. Для работы с приватными каналами используйте ID канала (начинается с -100)
4. База данных SQLite создается автоматически при первом запуске

## 🔐 Безопасность

- Храните `.env` файл в секрете
- Не публикуйте токен бота в открытом доступе
- Добавьте `.env` в `.gitignore`

## 🐛 Решение проблем

**Бот не проверяет подписки:**
- Убедитесь, что бот является администратором в канале
- Проверьте правильность ID/username канала

**Ошибка при запуске:**
- Проверьте правильность токена в `.env`
- Убедитесь, что установлены все зависимости

**Розыгрыш не завершается автоматически:**
- Проверьте, что бот запущен
- Проверьте логи на наличие ошибок
